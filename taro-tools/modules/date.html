endTime

  • endTime(time: string | Date, formatStr: string, isEndTime: boolean, getAll: boolean): EndTimeKeys | string

  • 计算倒计时

    Parameters

    • time: string | Date

      根据类型传入不同的参数

    • formatStr: string

      d h m s ms 分别代表天、小时、分钟、秒、毫秒 将他们变为大写会补全为2位数 毫秒补全为3位数

    返回天时分秒对象 或者返回格式化后的时间字符串
